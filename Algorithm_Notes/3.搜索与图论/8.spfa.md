# spfa
## spfa求最短路

给定一个 n个点 m条边的有向图，图中可能存在重边和自环， 边权可能为负数。

请你求出 1号点到 n号点的最短距离，如果无法从 1号点走到 n号点，则输出 impossible。数据保证不存在负权回路。

输入格式

第一行包含整数 n和 m

接下来 m行每行包含三个整数 x,y,z，表示存在一条从点 x到点 y的有向边，边长为 z

输出格式

输出一个整数，表示 1号点到 n号点的最短距离。

如果路径不存在，则输出 impossible。

数据范围

1≤n,m≤10^5,图中涉及边长绝对值均不超过 10000

```
//spfa算法是bellman-ford算法的改进版，类似dijkstra算法
//bellman搜索法每次将所有节点都放入队列更新其他节点 
//spfa算法只是将上一次被更新过改变了dist值的节点放入队列去更新其他节点 
#include<iostream>
#include<cstring>
#include<algorithm>
#include<queue>

using namespace std;

const int maxn=1e5+10;
int e[maxn],ne[maxn],h[maxn],w[maxn];
bool vis[maxn];
int dist[maxn];

int n,m,idx;

void add(int x,int y,int z)
{
	w[idx]=z;
	e[idx]=y;
	ne[idx]=h[x];
	h[x]=idx++;
}

void spfa()
{
	memset(dist,0x3f,sizeof(dist));
	dist[1]=0;
	//和dijkstra算法一样，起始时候1节点还没有被使用
	//放入队列后再标记 
	
	queue<int> q;
	vis[1]=true;
	q.push(1);
	
	while(q.size()){
		int t=q.front();
		q.pop();
		
		vis[t]=false;//出队列标记为没有访问 
		for(int i=h[t];i!=-1;i=ne[i]){
			int j=e[i];
			if(dist[j]>dist[t]+w[i]){//像dijkstra一样更新 
				dist[j]=dist[t]+w[i]; 
				if(!vis[j]){
					vis[j]=true;
					q.push(j);
				}
			}
		}
	}
	if(dist[n]==0x3f3f3f3f){
		printf("impossible\n");
	}
	else{
		printf("%d\n",dist[n]);
	}
}
 


int main()
{
	memset(h,-1,sizeof(h));
	scanf("%d%d",&n,&m);
	
	for(int i=1;i<=m;i++){
		int a,b,c;
		scanf("%d%d%d",&a,&b,&c);
		add(a,b,c);
	}
	
	spfa();
	
	return 0;
}
```



## spfa判断负环

给定一个 n个点 m条边的有向图，图中可能存在重边和自环， 边权可能为负数。

请你判断图中是否存在负权回路。

输入格式

第一行包含整数 n和 m

接下来 m行每行包含三个整数 x,y,z，表示存在一条从点 x到点 y的有向边，边长为 z

输出格式

如果图中存在负权回路，则输出 Yes，否则输出 No。

数据范围

1≤n≤2000,1≤m≤10000,图中涉及边长绝对值均不超过 10000


```
#include<iostream>
#include<cstring>
#include<algorithm>
#include<queue>

using namespace std;

const int maxn=1e5+10;
int e[maxn],ne[maxn],h[maxn],w[maxn];
bool vis[maxn];
int dist[maxn];
int cnt[maxn];

int n,m,idx;

void add(int x,int y,int z)
{
	w[idx]=z;
	e[idx]=y;
	ne[idx]=h[x];
	h[x]=idx++;
}

bool spfa()
{
    //仅仅判断负环不需要将所有路径设置为最大
    //也就是正常更新节点完毕后，判断标准不需要用某个dist为最大值来判断

	queue<int> q;
	//判断负环要把所有点先加入队列 
	for(int i=1;i<=n;i++){
		vis[i]=true;
		q.push(i);
	}
	
	while(q.size()){
		int t=q.front();
		q.pop();
		
		vis[t]=false;//出队列标记为没有访问 
		for(int i=h[t];i!=-1;i=ne[i]){
			int j=e[i];
			if(dist[j]>dist[t]+w[i]){//像dijkstra一样更新 
				dist[j]=dist[t]+w[i]; 
				cnt[j]=cnt[t]+1;
				//判断总边数
				//如果边数大于n-1说明有n个以上节点，也就是必定存在负环 
				if(cnt[j]>=n){
					return true;
				}
				if(!vis[j]){
					vis[j]=true;
					q.push(j);
				}
			}
		}
	}
	return false;
}
 


int main()
{
	memset(h,-1,sizeof(h));
	scanf("%d%d",&n,&m);
	
	for(int i=1;i<=m;i++){
		int a,b,c;
		scanf("%d%d%d",&a,&b,&c);
		add(a,b,c);
	}
	
	if(spfa()){
		printf("Yes\n");
	}
	else{
		printf("No\n");
	}
	
	return 0;
}
```
