# 最坏运气

## 耐摔指数
https://www.lanqiao.cn/courses/18397/learning/

X 星球的居民脾气不太好，但好在他们生气的时候唯一的异常举动是：摔手机。

各大厂商也就纷纷推出各种耐摔型手机。X 星球的质监局规定了手机必须经过耐摔测试，并且评定出一个耐摔指数来，之后才允许上市流通。

X 星球有很多高耸入云的高塔，刚好可以用来做耐摔测试。塔的每一层高度都是一样的，与地球上稍有不同的是，他们的第一层不是地面，而是相当于我们的 2 楼。

如果手机从第 7 层扔下去没摔坏，但第 8 层摔坏了，则手机耐摔指数 = 7。

特别地，如果手机从第 1 层扔下去就坏了，则耐摔指数 = 0

如果到了塔的最高层第 
n 层扔没摔坏，则耐摔指数 = n。

为了减少测试次数，从每个厂家抽样 3 部手机参加测试。

如果已知了测试塔的高度，并且采用最佳策略，在最坏的运气下最多需要测试多少次才能确定手机的耐摔指数呢？

```
#include<iostream>
using namespace std;

const int maxn=1e5+10;
int f[4][maxn];

//f[i][j]表示有i部手机楼数为j的情况下最少测试次数
//在第k层测试 
//f[i][j]有两种情况
//手机摔坏了，手机数量减一，下一次去楼下测试，最多k-1次
//f[i-1][k-1]+1
//手机没摔坏，手机数量不变，下一次去楼上测试，最多试j-k次
//f[i][j-k]+1

int main()
{
	int n;
	cin>>n;
	
	for(int i=1;i<=3;i++){
		for(int j=1;j<=n;j++){
			f[i][j]=j;
            //初始化，不管什么情况的最坏运气就是j，其实最主要还是第一层的初始化
		}
	}
    //min模拟最好策略，max模拟最坏运气
	for(int i=2;i<=3;i++){
		for(int j=1;j<=n;j++){
			for(int k=1;k<j;k++){
				f[i][j]=min(f[i][j],max(f[i-1][k-1],f[i][j-k])+1);
			}
		}
	} 
	printf("%d\n",f[3][n]);
	
	
	return 0;
}
```
