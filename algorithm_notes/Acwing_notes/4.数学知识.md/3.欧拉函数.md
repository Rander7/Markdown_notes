# 欧拉函数

## 欧拉函数

给定 n个正整数 ai，请你求出每个数的欧拉函数。

1~N中与 N互质的数的个数被称为欧拉函数，记为 f
(N)。

$$ N=pa^1pa^2…pa^m
 $$

$$ f(N)= N×(p1-1)/p1×(p2-1)/p2×…×(pm-1)/pm$$


输入格式

第一行包含整数 n。

接下来 n行，每行包含一个正整数 ai
。

输出格式

输出共 n行，每行输出一个正整数 ai的欧拉函数


```
#include<iostream>
#include<algorithm>

using namespace std;

int main()
{
	int n;
	scanf("%d",&n);
	
	while(n--)
	{
		long long x;
		scanf("%lld",&x);
		long long res=x;
		for(long long i=2;i<=x/i;i++){
			if(x%i==0){
                //避免出现小数
				res=res/i*(i-1);				
				while(x%i==0){
					x/=i;
				}
			}
		}
		if(x>1){
			res=res/x*(x-1);
		}	
		printf("%lld\n",res);
	}

	return 0;
}
```

## 筛法求欧拉函数

给定一个正整数 n，求 1~n中每个数的欧拉函数之和。

输入格式

共一行，包含一个整数 n。

输出格式

共一行，包含一个整数，表示 1~n中每个数的欧拉函数之和。

数据范围

1≤n≤10^6


```

#include<iostream>
using namespace std;

const int N=1000010;
int n;
int primes[N];
int vis[N];
int cnt;
int phi[N];
long long res;

void eurlus_primes(int n)
{
	//线性筛的第一步要看完全部n个点,后面才优化n/i 
	for(int i=2;i<=n;i++)
	{
		if(!vis[i]){
			primes[cnt++]=i;
			//从1到质数i与其互质的数个数是i-1
			phi[i]=i-1; 
		}
		
		for(int j=0;primes[j]<=n/i;j++){
			vis[primes[j]*i]=true;
			
			//但primes[j]是i的一个约数时候
			//i*primes[j]和primes[j]的质因数完全相同 
			//欧拉函数项中多了primes[j] 
			if(i%primes[j]==0){
				phi[i*primes[j]]=primes[j]*phi[i];
				break;
			}
			// 此时欧拉函数项中多了primes[j]*(1-1/primes[j]) 
			phi[i*primes[j]]=(primes[j]-1)*phi[i];
		}
	}
	
}



int main()
{
	scanf("%d",&n);
	phi[1]=1;//与1互质的只有1,欧拉函数表示与i互质数的个数
	eurlus_primes(n);
	for(int i=1;i<=n;i++){
		res+=phi[i];
	} 
	printf("%lld",res);
	
	return 0;
}

```
