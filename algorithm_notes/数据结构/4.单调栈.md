# 单调栈

**栈中可以存放下标也可以存放元素值**

## 单调栈
给定一个长度为 N的整数数列，输出每个数左边第一个比它小的数，如果不存在则输出 -1

输入格式

第一行包含整数 N，表示数列长度。

第二行包含 N个整数，表示整数数列。

输出格式

共一行，包含 N个整数，其中第 i个数表示第 i个数的左边第一个比它小的数，如果不存在则输出 -1

```
//单调栈

#include<iostream>
using namespace std;

const int maxn=1e5+10;

int a[maxn];
int sta[maxn];
int n;

int main()
{
	int tt=0;
	scanf("%d",&n);
	for(int i=0;i<n;i++){
		scanf("%d",&a[i]);
	}
	
	for(int i=0;i<n;i++){
		//如果栈不为空且新元素小于等于栈顶元素，栈顶元素在未来无效，出栈
		while(tt&&a[sta[tt]]>=a[i]){
			tt--;
		} 
		if(!tt){
			printf("-1 ");
		}
		else{
			printf("%d ",a[sta[tt]]);
		}
		sta[++tt]=i;
	}
	
	return 0;
} 
```
