# 堆

## 堆排序
输入一个长度为 n的整数数列，从小到大输出前 m小的数。

输入格式

第一行包含整数 n和 m

第二行包含 n个整数，表示整数数列。

输出格式

共一行，包含 m个整数，表示整数数列中前 m小的数。

数据范围

1≤m≤n≤10^5，1≤数列中元素≤10^9

```
#include<iostream>

using namespace std;

const int maxn=1e5+10;
int n,m,size1;

int h[maxn];

void down(int x)
{
	int t=x;
	if(2*x<=size1&&h[t]>h[2*x]){
		t=2*x;
	}
	if(2*x+1<=size1&&h[t]>h[2*x+1]){
		t=2*x+1;
	}
	if(t!=x){
		swap(h[t],h[x]);
		down(t);
	}
}

int main()
{
	cin>>n>>m;
	size1=n;
	
	for(int i=1;i<=n;i++){
		scanf("%d",&h[i]);
	}
	
	//先把堆排好序
	for(int i=n/2;i;i--){
		down(i);
	} 
	
	while(m--)
	{
		printf("%d ",h[1]);
		//把根节点删除再排序，根节点就是第二小的数		 
		h[1]=h[size1];
		size1--;
		down(1);		
	}

	return 0;
}
```
